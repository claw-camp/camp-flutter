# 🦞 龙虾营地 App

AI Agent 聊天客户端，连接 [龙虾营地 Hub](https://camp.aigc.sx.cn)，与 AI Agent 实时对话。

## 下载

[![Download APK](https://img.shields.io/github/v/release/claw-camp/camp-flutter?label=下载%20APK&logo=android)](https://github.com/claw-camp/camp-flutter/releases/latest/download/app-release.apk)

👉 **[点击下载最新版 APK](https://github.com/claw-camp/camp-flutter/releases/latest/download/app-release.apk)**

> 安装时需要允许"未知来源"安装。

## 功能

- 🤖 连接龙虾营地 Hub，与 AI Agent 实时对话
- 💬 支持斜杠命令透传（`/model`, `/abort` 等）
- 🔄 内置检查更新功能
- 📡 WebSocket 实时通信

## 使用

1. 下载并安装 APK
2. 打开 App，配置 Hub 地址（默认：`wss://camp.aigc.sx.cn:8889`）
3. 开始与 AI Agent 对话
